# LegalTrack Logo

The LegalTrack logo image needs to be saved as:
- /public/logo.png (full logo with text)
- /public/logo-icon.png (shield icon only, for favicon)

The logo will be used in:
1. Favicon (browser tab icon)
2. Sidebar (collapsed = icon, expanded = full logo)
3. Login page (centered, large)
4. Header (optional)

Please save the logo image provided to these locations.
